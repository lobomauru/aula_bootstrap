#### Linguagens de Programação;
- ###### Termos comuns;
- ALT - Texto alternativo gerelmente usado para descritivo em links e imagens;
- API - Ferramenta direcionada à mover parte do site para um ambiente externo para realizar alguma função específica. Ex. Caixa de login do Google/facebook para logar em site de compras sem a necessidade de cadastramento;
- Biblioteca/dll - Conjunto de ferramentas para novas funcionalidades;
- Boostrap - Framework de JavaScrip;
- Breakpoint - Pontos de parada para análise e validação de parte do código.
- Deploy - Publicação de parte do código;
- DPI - Proporção de pontos na tela (resolução);
- Favicon - Icone do site que está sendo visitado;
- Meta tags - Tags introduzidas nos headers do sites;
- MVC - Model Viewer Controler - Arquitetura utilizada para separar a estrutura dos dados, a visualização e a lógica alicada no site; 
- MVP - Minimum Viable Product - Produto disponibilizado com os mínimos recursos com intuito de testes de viabilidade;
- SSL - Camada de segurança dos sites;
- UI - User interface;
- UX - User experience;
- WYSIWYG - What You See Is What You Get - Edição de textos e imagens cuja formatação é vista da mesma maneira que a programada.
